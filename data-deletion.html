Data Deletion Instructions

Last updated: 2026-05-20

“The App” (Hermes Personal Digest) is a single-user personal tool. It does not store personal data belonging to other people. Public Threads posts are processed transiently to produce a private digest for the app owner and are not retained in any shared dataset.

How to request deletion

Because there is no shared user dataset, there is nothing to delete on behalf of third parties. The app owner can, at any time:

  • Delete the optional local digest archive stored on the owner's private server.
  • Revoke the App's access token via the Meta App Dashboard (Apps → Settings), which immediately stops all data access.
